คัดลอกข้อมูลประวัติการเรียนรู้
เตรียมย้ายเข้า Hermes Memory

หน้านี้ช่วยสร้างกรอบคำถาม (Prompt Template) ในการนำเข้าข้อมูลและเป้าหมายการประเมินทักษะ เพื่อให้ AI chatbot ของท่าน (ChatGPT / Claude / Gemini) สรุปและประมวลผลออกมาเป็นแฟ้มประวัติการเรียนรู้สำหรับป้อนเข้าฐานข้อมูลความจำของ Hermes Agent ต่อไป

วิธีการใช้งาน

  • คลิกปุ่ม "คัดลอก Prompt" ด้านบนเพื่อบันทึกเทมเพลตคำถามลงคลิปบอร์ด
  • นำไปวาง (Paste) ใน Chatbot ตัวโปรดของท่าน (เช่น ChatGPT, Claude หรือ Gemini)
  • ตอบคำถามเกี่ยวกับเป้าหมายและสไตล์การเรียนทีละหมวดตามหัวข้อ
  • ให้ Chatbot ทำการประมวลผลและสรุปออกมาเป็นรูปแบบ Markdown หรือ PDF
  • นำไฟล์ประวัติที่ได้ส่งเข้าระบบหรือแจ้งทีมงานสภาเพื่อจัดส่งเข้า Hermes Agent ต่อไป

คำเตือนความปลอดภัย

  • ห้ามกรอกข้อมูลลับเด็ดขาด: เช่น รหัสผ่าน, API Token, Private Key, เลขบัตรประชาชน, ข้อมูลบัตรเครดิต หรือความลับบริษัท
  • ประวัติในหน้านี้ควรเน้นเฉพาะ "ประวัติทักษะและแนวทางการเรียนรู้" เท่านั้น เพื่อให้ระบบประเมินผลการเรียนรู้ได้อย่างถูกต้อง
  • ตรวจสอบให้แน่ใจว่าไม่มีข้อความอ่อนไหวปนเข้าไปก่อนการบันทึก PDF
